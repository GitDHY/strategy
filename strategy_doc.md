# 动态宏观资产配置策略文档 (Dynamic Macro Asset Allocation Strategy) v1.4

## 1. 策略概述
本策略基于宏观经济数据的"状态机"模型，自动分析当前的经济环境（如衰退、通胀、恐慌等），并将市场划分为不同的"经济季节"。根据当前季节，策略会自动调整各类资产（股票、债券、黄金、现金等）的配置比例，以实现"危机时防御，牛市中进攻"的目标。

## 3. 宏观状态定义 (Macro States)

策略定义了以下 6 种宏观状态，每种状态对应不同的风险偏好：

### 🔴 滞胀 / 加息冲击 (INFLATION_SHOCK)
*   **描述**: 利率飙升或出现股债双杀。传统资产失效。
*   **策略**: **现金为王**。清仓长久期资产（如长债、成长股），转入趋势跟踪策略 (WTMF) 或短债现金。

### 🔵 衰退 / 崩盘 (DEFLATION_RECESSION)
*   **描述**: 经济衰退确认或流动性危机。
*   **策略**: **全面防御**。锁定国债收益 (TLT/MBH)，配置黄金避险，保留少量防御性权益。

### 🚀 极度贪婪 / 抄底 (EXTREME_ACCUMULATION)
*   **描述**: 市场极度恐慌 (VIX高企) 但基本面未崩坏（无衰退/无加息冲击）。
*   **策略**: **重仓抄底**。利用别人的恐慌获利，大幅买入成长股 (IWY)。

### ⚠️ 谨慎 / 趋势破位 (CAUTIOUS_TREND)
*   **描述**: 长期趋势转空（价格跌破200日均线），但尚未出现极度恐慌（阴跌状态）。
*   **策略**: **防御配置**。减少成长股暴露，重仓红利低波 (LVHI) 与现金类资产。

### ⚡ 谨慎 / 高波震荡 (CAUTIOUS_VOL)
*   **描述**: 趋势尚可，但波动率加剧 (VIX > 20)。
*   **策略**: **对冲配置**。保留成长仓位，但大幅增加危机Alpha (WTMF) 进行对冲。

### 🟢 常态 / 牛市 (NEUTRAL)
*   **描述**: 市场运行平稳，波动率低且趋势向上。
*   **策略**: **标准增长配置**。维持较高的成长股仓位，享受复利增长。**启用双重动量过滤**。

## 4. 状态判定逻辑 (State Logic)

系统根据以下指标判定当前状态（优先级从上到下）：

1.  **INFLATION_SHOCK (滞胀/加息)**
    *   触发条件: `RateShock > 0.20` (长债收益率飙升) **或** (`Sahm >= 0.50` 且 `Corr > 0.3`) (衰退且股债相关性失效/正相关)。
2.  **DEFLATION_RECESSION (衰退)**
    *   触发条件: `Sahm >= 0.50` (萨姆规则触发) **或** (`Trend_Bear` 且 `VIX > 35`) (熊市且极度恐慌)。
3.  **EXTREME_ACCUMULATION (抄底)**
    *   触发条件: `VIX > 32` (恐慌指数高) 且 **非**冲击 **非**衰退。
4.  **CAUTIOUS_TREND (趋势破位)**
    *   触发条件: `Trend_Bear` (价格 < 200日均线) 且 `VIX <= 20` (波动率不高)。
5.  **CAUTIOUS_VOL (高波震荡)**
    *   触发条件: `VIX > 20` (波动率偏高)。
6.  **NEUTRAL (常态)**
    *   触发条件: 以上皆不满足。

**核心指标说明**:
*   **Sahm (萨姆规则)**: 失业率3个月移动平均 - 过去12个月最低值。 >= 0.50 预示衰退。
*   **RateShock**: 10年期美债收益率 (TNX) 的21日变化率。
*   **Trend_Bear**: 标的 (IWY) 价格低于 200日均线。
*   **Corr (股债相关性)**: IWY 与 TLT 的 60日滚动相关系数。

## 6. 战术微调 (Tactical Modifiers)

在上述基础配置之上，策略还会根据特定指标进行微调：

### 1. 黄金趋势过滤 (Gold Trend Filter)
*   **适用状态**: 所有状态
*   **逻辑**: 如果黄金价格低于 200日均线 (`Gold < MA200`)，视为黄金处于熊市。
*   **操作**: **清仓** 黄金 (GSD.SI)，资金转移至 **WTMF**。

### 2. 风格轮动 (Value Regime)
*   **适用状态**: NEUTRAL, CAUTIOUS_TREND
*   **逻辑**: 比较成长股(IWY)与价值股(IWD)的比价趋势。如果比价跌破200日均线，视为价值风格占优 (Value Regime)。
*   **操作**:
    *   **NEUTRAL 状态**: IWY 降至 45%，LVHI 升至 20%。
    *   **CAUTIOUS_TREND 状态**: IWY 降至 5%，LVHI 升至 25%。

### 3. 全局动态趋势熔断 (Global Dynamic Trend Filter)
*   **适用状态**: **除 EXTREME_ACCUMULATION (抄底) 外的所有状态**
*   **逻辑**: 任何资产如果处于熊市趋势 (Price < MA200)，都应该被削减。
*   **检查列表**: `G3B.SI`, `LVHI`, `MBH.SI`, `GSD.SI`, `SRT.SI`, `AJBU.SI`
*   **操作**:
    *   **清仓**: 若某资产价格 < MA200，强制清仓该资产。
    *   **资金再分配**:
        *   在 **NEUTRAL (牛市)** 状态下：如果 **IWY (成长)** 强势，资金流向 **IWY**；否则流向 **WTMF**。
        *   在 **其他状态** 下：资金流向 **WTMF**。

### 4. 核心资产熔断 (Core Asset Safety Valve)
*   **适用状态**: **除 EXTREME_ACCUMULATION 外**
*   **逻辑**: 如果核心成长资产 **IWY** 跌破 200 日均线。
*   **操作**: IWY 仓位 **削减 50%** (若 VIX > 25 则削减 **80%**)，资金移至 **WTMF**。

### 5. 动态风控层 (Dynamic Risk Control) v1.4
*   **适用状态**: NEUTRAL, DEFLATION_RECESSION, CAUTIOUS_TREND
*   **A. 波动率杠杆 (VIX Booster)**
    *   **极度平稳期 (VIX < 13)**: **加仓成长**。WTMF 清仓转入 IWY，MBH 减半转入 IWY。
    *   **早期动荡预警 (VIX > 20)**: **提前减仓**。IWY 减仓 20% 转入 WTMF。
*   **B. 收益率曲线防御 (Yield Curve Guard)**
    *   **逻辑**: 处于衰退/震荡期且收益率曲线深度倒挂 (10Y-2Y < -0.30%)。
    *   **操作**: **MBH.SI 削减 70%**，资金移至 WTMF。

## 7. 新手指南：市场状态与应对策略 (Beginner's Guide)

### 快速上手路径
1) **识别宏观状态**：先用"状态判定逻辑"确定当前属于 6 种状态之一。
2) **套用基础配置**：按"资产配置矩阵"得到初始目标仓位（v1.5: CAUTIOUS_VOL支持VIX分层）。
3) **检查动态风控触发**：逐条核对下表的触发条件，若满足则立即执行对应调整。
4) **应用新增优化模块**：按顺序检查双均线趋势、市场广度、Sahm预警、解倒挂保护、相关性渐进调整、VIX均值回归、现金缓冲。
5) **执行风格/趋势过滤**：应用黄金趋势过滤、风格轮动与全局趋势过滤，得到最终持仓。
6) **参考执行建议**：根据波动率、状态持续时间等决定执行节奏。

### 执行建议与节奏控制

#### 1. 信号确认机制
*   状态切换后建议观望 **2天** 确认，避免追涨杀跌。
*   若状态仅持续 1-2 天就切换回来，说明信号不稳，建议维持原配置。

#### 2. 波动率执行缩放
| VIX区间 | 建议执行比例 | 说明 |
| :--- | :---: | :--- |
| VIX < 15 | 100% | 正常执行 |
| 15 ≤ VIX < 30 | 60-100% (线性) | 波动偏高，留现金缓冲 |
| VIX ≥ 30 | 60% | 高波动期，仅执行60%目标仓位 |

#### 3. 分步过渡机制
*   配置变化 >20% 时，建议分 **3天** 逐步调整：
    *   **第1天**：执行所有卖出操作
    *   **第2天**：执行高优先级买入
    *   **第3天**：完成剩余调整

#### 4. 再平衡容忍带
*   单一资产偏离目标 <5% 时，可暂不调仓以节省交易成本。
*   建议每月检查一次，季度强制再平衡。

#### 5. 止损与恢复机制
*   持仓回撤超过 **-10%** 时，风险资产减仓 **50%**。
*   回撤恢复到 **-5%** 以内后方可恢复原配置。

### 常见问题 FAQ

**Q1: 多个触发器同时满足怎么办？**
A: 按代码执行顺序依次应用，后执行的会覆盖前面的调整。优先级：基础风控 → 优化层 → 风格轮动。

**Q2: 状态刚切换就要大幅调仓吗？**
A: 不建议。等待2天确认信号稳定，然后分3天逐步调整。

**Q3: 股债相关性失效提示了但MBH不够10%怎么办？**
A: 系统只在MBH>10%时才执行调整，若MBH本身≤10%则保持不变。

**Q4: 解倒挂保护期是什么意思？**
A: 历史数据显示收益率曲线从倒挂转正后9-18个月内，衰退风险反而最高。系统会在曲线转正后自动维持防御配置12个月。

**Q5: 再平衡多久做一次？**
A: 日常检查，但只有偏离>5%才调仓。建议至少每月检查，每季度强制再平衡。
